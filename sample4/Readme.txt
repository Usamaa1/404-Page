All the details of the project listed below:

CHANGE COLORS:
-You can Change colors by going to CSS file attached with each 404 Page document and then go to the very first line of ":root" pseudo-class in that you can find/change all the colors used on particular 404 page.

CHANGE TEXT:
-You can change the text of the page by going to HTML file attached to each 404 Page document and then change the text by going through the comments before the tags.

These all 404 pages are responsive and works on all big and small screens.
